# Guide Naming Rules

All guide markdown files in `docs/` must use this pattern:

`SqlLens_<GuideName>_Guide.md`

## Canonical guide filenames

- `SqlLens_Quick_Start_Guide.md`
- `SqlLens_Concise_Guide.md`
- `SqlLens_Reference_Guide.md`
- `SqlLens_Search_Guide.md`
- `SqlLens_Git_Integration_Guide.md`
- `SqlLens_Project_Management_Guide.md`
- `SqlLens_User_Guide_Full.md`
- `SqlLens_VSCode_Guide.md`
- `SqlLens_Agent_Jobs_Guide.md`

## Rules

1. Do not create non-prefixed guide files (for example `Quick_Start_Guide.md`).
2. Keep Desktop and `sqllens-docs` guide names aligned.
3. Update links when adding or renaming a guide.
